# Чек-лист тестирования виджета Power Manager (до 25 запусков)

Перед использованием выполните не менее 10–25 запусков и проверьте пункты ниже. Цель: 25 тестов на 10/10.

## Запуск

```powershell
dotnet run --project src\PowerManagerWidget\PowerManagerWidget.csproj
```

или

```powershell
.\src\PowerManagerWidget\bin\Release\net8.0-windows\PowerManagerWidget.exe
```

---

## 1. Батарея

- [ ] Строка **«Батарея»** всегда видна (под кнопками режимов и «Авто»).
- [ ] **Процент заряда** отображается всегда: при питании от сети — «X% (от сети)», при разрядке/зарядке — «X%, ~время».
- [ ] При работе от сети: «X% (от сети)» (не только «от сети»).
- [ ] При разрядке: «X%, ~N ч M мин до разрядки» (или только минуты).
- [ ] При зарядке: «X%, ~N ч M мин до полной зарядки».
- [ ] Строка не склеивается с ползунком «Яркость».

## 2. Энергосбережение и системная панель Windows

- [ ] Галочка **«Энергосбережение»** в виджете **включает** системное энергосбережение (переключатель в панели быстрых настроек Win+A).
- [ ] Снятие галочки **выключает** системное энергосбережение.
- [ ] При загрузке виджета галочка синхронизирована с системой (если в системе включено — галочка отмечена).

## 3. Галочки и нижний блок без склеивания

- [ ] **Энергосбережение** — отдельная строка с отступом.
- [ ] **Поверх всех окон** — отдельная строка, текст читается полностью (не «Поверхност»).
- [ ] Блоки **«Графики»** и **«Проверка авто»** внизу видны, выпадающие списки работают.
- [ ] Высота окна достаточна: все элементы внизу видны без наложения.

## 4. Графики и данные

- [ ] ЦП: частота (ГГц), нагрузка (%), температура (°C).
- [ ] На графиках есть сетка.

## 5. Режимы и авто

- [ ] Три кнопки режимов переключают схему.
- [ ] Моргающий индикатор рекомендуемого режима виден.
- [ ] «Авто» переключает режим по нагрузке/температуре/батарее.

## 6. Яркость и звук

- [ ] Ползунок яркости и звука работают, значения (%) отображаются.

## 7. Общее

- [ ] Окно перетаскивается; «Поверх всех окон» работает.
- [ ] После 10–25 запусков нет зависаний и ошибок.
